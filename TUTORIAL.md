### Primeiro programa

Primeiro precisamos fazer o mais simples: compilar um exemplo de SDL somente com C/C++

O código é muito simples:

```
int main (void) {

}
```

Para compilar, basta escrever os seguintes comandos:

```
$ g++ main.c
$ ./a.out
```
